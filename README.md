# 📝 TaskMaster - Django To-Do App

A sleek and functional task management web application built with Django. This project was created to manage daily tasks with a focus on user authentication and clean UI.

## ✨ Features
- User Authentication: Secure login and logout system.
- Personalized Lists: Each user sees only their own tasks.
- Task Management (CRUD): Add, view, and delete tasks.
- Toggle Feature: Mark tasks as complete/incomplete with a single click.
- Responsive Design: Modern UI with custom CSS.

## 🚀 Tech Stack
- Framework: Django 6.0
- Database: SQLite
- Styling: Custom CSS
- Language: Python

## 🛠️ How to run locally
1. Clone the repository.
2. Activate your virtual environment: .\venv\Scripts\activate
3. Install requirements: pip install -r requirements.txt
4. Run migrations: python manage.py migrate
5. Start server: python manage.py runserver
